# Studio Ghibli Film Browser

## Overview
This project displays Studio Ghibli films by year using a public API. Users can enter a year to view information about films released by the beloved Japanese animated studio that year, whether their release year matches a birth year or an anniversary. Users can also enter 0000 to list all films released up to 2022.
Information includes the director, the Rotten Tomatoes score, a brief plot description, a poster, and more. All data was sources by by https://ghibliapi.vercel.app/ and TMDB.

## Technologies Used
- React
- HTML
- CSS
- JavaScript
- Studio Ghibli API
